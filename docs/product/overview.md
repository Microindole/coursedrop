# 产品说明

CourseDrop，中文名“课递”，是一个面向课程小组的资料快传与临时存储工具。

它不是聊天软件，也不是网盘。项目重点是帮助同一课程小组在教室、宿舍、实验室等场景下快速分发文件、图片、链接和剪贴板片段，并通过服务器提供限时中转与自动清理。

## 核心定位

```text
课程小组资料协作工具 = 临时房间 + 文件快传 + 剪贴板片段 + 限时存储
```

## 不做什么

- 不做好友系统
- 不做私聊和群聊
- 不做长期社交关系
- 不做永久网盘
- 不做动态、评论区、朋友圈类功能

## 典型场景

- 组长创建一个“移动应用开发第 3 组”房间。
- 组员通过 6 位码或二维码加入。
- 成员上传作业文档、图片、压缩包、代码片段或链接。
- 服务器保存资料一段时间，到期自动清理。
- 后续版本在同一局域网下优先直连传输，跨网络时再走服务器中转。

## 项目亮点

- 鸿蒙原生 ArkTS / ArkUI 客户端
- Java Spring Boot 轻量后端
- 房间码加入，弱化账号和社交属性
- 文件与剪贴板统一为传输项
- 服务器限时存储，避免变成普通网盘
- 后续可扩展 WebSocket、二维码、局域网直传和桌面卡片
