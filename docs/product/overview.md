# 产品说明

CourseDrop，中文名“课递”，是一个本地优先的资料快传、临时中转与可视化分享管理工具。

它不是聊天软件，也不是网盘。项目重点是用手机端管理本地内容，在局域网内优先传输，跨网络时通过服务器提供限时中转、二维码下载和自动清理。

## 核心定位

```text
本地优先文件传输工具 = 本地库 + 局域网直传 + 公网限时中转 + 浏览器下载
```

## 不做什么

- 不做好友系统
- 不做私聊和群聊
- 不做长期社交关系
- 不做永久网盘
- 不做动态、评论区、朋友圈类功能

## 典型场景

- 用户在手机 CourseDrop 中选择本地文件、图片、文本或链接。
- 同一局域网设备优先直连传输。
- 跨网络分享时，手机上传到公网中转服务器并生成二维码。
- 接收方可以用 CourseDrop App 下载，也可以用任意二维码工具扫码后在浏览器中下载。
- 服务器只在有效期内保存内容，到期或撤回后删除。

## 项目亮点

- 鸿蒙原生 ArkTS / ArkUI 客户端
- Java Spring Boot 轻量后端
- 二维码和浏览器下载，降低接收方安装门槛
- 文件与剪贴板统一为传输项
- 服务器限时存储，避免变成普通网盘
- 设备指纹与账号绑定，支持手机扫码登录 Web 管理端
- 后续可扩展 WebSocket、局域网发现和桌面卡片
