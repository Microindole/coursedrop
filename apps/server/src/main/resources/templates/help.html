使用指南

了解 CourseDrop 的工作方式与安全特性

什么是限时文件中转？

中转文件是通过高级加密算法临时保存在服务器上的。一旦到达设定的过期时间，或被所有者手动点击“撤销”，中转文件将从服务器的存储介质上进行**物理粉碎删除**，不可找回，确保数据隐私安全。

服务限制规则

  • 单文件体积上限：512 MB
  • 默认存储有效期：24 小时
  • 支持端到端加密传输（网页端本地基于 WebCrypto 流式解密）

安全提取策略定义

公开提取 (PUBLIC)

任何获取到该通道分享码的客户端都可免登录直接提取并解密下载。

限时锁屏 (LOGIN_REQUIRED)

浏览器端必须通过手机客户端扫码，或者使用绑定的账号密码验证登录后，方可解锁提取通道。

仅所有者 (OWNER_ONLY)

该提取码仅限发出分享的设备指纹或者该设备所绑定的主账户下的关联设备提取，其他设备无法解锁。

极客快捷快捷跳转指令

本站搜索框支持命令行风格的快捷指令。在首页提取输入框中，以 ! 开头输入对应路径即可一键直达隐藏调试或检测页面：

  • !/health → 直达后端运行健康状态检测
  • !/caps → 直达后端服务核心能力清单
